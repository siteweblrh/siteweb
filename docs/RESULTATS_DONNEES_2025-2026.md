## 8. Format JSON (pour import automatique)

Tous les résultats sont disponibles au format JSON exploitable à l'emplacement suivant (script qui interroge la DB HCO) :

```bash
node scripts/dump-results.mjs > resultats-2025-2026.json
```

Structure d'un objet match :

```json
{
  "date": "2025-09-21T05:00:00.000Z",
  "label": null,
  "location": "Complexe Gymnase Saint Paul",
  "home": "USPG",
  "homeScore": 5,
  "away": "HCO",
  "awayScore": 7
}
```

> ⚠️ Les dates sont en **UTC**. À la Réunion (UTC+4), il faut **ajouter 4 heures**. Exemple : `2025-09-21T05:00:00Z` = `21/09/2025 à 09:00` heure locale.

## 9. Notes pour l'import dans le site Ligue

1. **Saison** : utiliser le format `"2025-2026"` (cohérent avec ce qui existe en base HCO).
2. **Lieux à standardiser** :
   - `Complexe Gymnase Saint Paul` → `Complexe Gymnase Saint-Paul`
   - `Stade de Manès` → vérifier orthographe (Manes / Mannès)
   - `Complexe de Bois Rouge - Ravine à Malheur` → préfère `—` (tiret cadratin)
3. **Ententes à créer en priorité** :
   - HCP/HCD (gazon) — logos HCP + HCD
   - SDHC/HHS (salle) — logos SDHC + HHS
4. **Aucune donnée buteur** n'a été saisie pour cette saison sur HCO. À récupérer auprès des clubs si la ligue veut une rubrique "Top buteurs".
5. **Compétitions à compléter** côté Ligue après import :
   - Coupe de la Réunion Gazon (dates à venir)
   - Compétitions jeunes (U11, U13, U15, U17) si la ligue les organise — pas dans la base HCO.
